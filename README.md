# Azeroth Hold'em

Peer-verifiable Texas Hold'em for World of Warcraft 3.3.5a (WotLK), with real card art
and a cryptographic anti-cheat layer.

## Install
1. Get this folder named exactly **`AzerothHoldem`** (a `git clone` already names it
   correctly; if you downloaded a ZIP, rename the extracted folder to `AzerothHoldem`).
2. Place it in your WoW directory under `Interface\AddOns\AzerothHoldem\`
   (so that `Interface\AddOns\AzerothHoldem\AzerothHoldem.toc` exists).
3. Restart WoW (or `/reload`) and enable **Azeroth Hold'em** at the character select
   AddOns screen.

## Use
`/azh` lists commands — host or join a table, open the casino lobby, and play.
